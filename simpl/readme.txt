A PHP framework for rapidly creating classes that mirror MySQL tables. The framework comes with the basic functions to list, display and edit records from the database. With this framework a simple manager and front of a site can be created within a few minutes. An example database, manager and front end are included with the framework.

Features
* Database abstraction
* Automatically configured database functions (ie. GetInfo, GetList, Save, Delete, Search and Form)
* Advanced automatic form creation to mirror database/class expectations
* Caching
* Form validation
* Upload, Move and Copy Files
* Resize and Rotate Images
* Email with class abstraction
* RSS0.91, RSS1.0, RSS2.0, ATOM Feeds
* XML, CSV Exports
